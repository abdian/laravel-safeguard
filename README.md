# Laravel Safeguard

A Laravel security package for safeguarding your application.

## Installation

Install the package via Composer:

```bash
composer require abdian/laravel-safeguard
```

## Configuration

Publish the configuration file:

```bash
php artisan vendor:publish --tag=safeguard-config
```

This will create a `config/safeguard.php` file where you can customize the package settings.
